# Fly-Over Pilot - Consolidated Assets

This folder contains the consolidated assets for the Fly-Over Pilot launch (January 2026).

## Projects

### 1. Vision Plan App (`/vision-app`)
An interactive React/Vite presentation of the "Visieplan" for innovation in driving instruction.
- **Branding**: Standardized Fly-Over branding.
- **Narrative**: High-impact, strategic partnership focus.

### 2. Pilot Dashboard (`/dashboard`)
The rebranded EduCapture dashboard tailored for the Fly-Over pilot.
- **Branding**: Fully white-labeled for Fly-Over (Navy Blue, Montserrat typography).
- **Setup**: Configured with the instructor "Fabian van Beek".

## How to Run

Each project is a standalone Vite application. To start a dev server:

### Vision Plan App
```bash
cd vision-app
npm install
npm run dev
```

### Pilot Dashboard
```bash
cd dashboard
npm install
npm run dev
```
